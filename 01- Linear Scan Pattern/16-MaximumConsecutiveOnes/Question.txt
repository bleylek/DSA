Maximum Consecutive Ones

Given a binary array consisting only of 0s and 1s, return the maximum number of consecutive 1s.

Example 1
Input:
[1, 1, 0, 1, 1, 1]

Output:
3

Explanation:

The longest consecutive sequence of 1s is [1, 1, 1].


Example 2
Input:
[1, 0, 1, 1, 0, 1]

Output:
2


Example 3
Input:
[0, 0, 0]

Output:
0


Constraints

Traverse the array only once.
Do not use nested loops.


Expected Complexity

Time Complexity: O(n)
Space Complexity: O(1)
