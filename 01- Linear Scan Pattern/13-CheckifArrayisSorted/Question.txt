Check if Array is Sorted

Given an integer array, determine whether the array is sorted in non-decreasing order.

Return true if the array is sorted; otherwise, return false.

Example 1
Input:
[1, 2, 3, 4, 5]

Output:
true

Example 2
Input:
[1, 2, 2, 4, 5]

Output:
true

Explanation:

The array is sorted because equal adjacent elements are allowed.

Example 3
Input:
[1, 5, 3, 4]

Output:
false

Explanation:

The array is not sorted because 5 > 3.

Example 4
Input:
[10]

Output:
true

Explanation:

A single-element array is always considered sorted.
