Count Occurrences of a Given Number

Given an integer array and a target value, return the number of times the target appears in the array.

Example 1
Input:
array = [1, 2, 3, 2, 4, 2, 5]
target = 2

Output:
3


Example 2
Input:
array = [7, 7, 7, 7]
target = 7

Output:
4


Example 3
Input:
array = [1, 3, 5]
target = 2

Output:
0

Constraints

Traverse the array only once.
Do not use nested loops.
Do not use any built-in counting functions.


Expected Complexity

Time Complexity: O(n)
Space Complexity: O(1)
