Reverse String

Given a string, reverse it in-place.

Example 1
Input:
"hello"

Output:
"olleh"


Example 2
Input:
"abcd"

Output:
"dcba"


Example 3
Input:
"a"

Output:
"a"


Constraints

Reverse the string in-place.
Do not create another string.
Do not use built-in functions such as reverse().


Expected Complexity

Time Complexity: O(n)
Space Complexity: O(1)
