I used the linear scan pattern for this question. In other words I visited all the elements via loop and gathered information.

First, I declared two variables for holding the current maximum element in the given array and for the index of the maximum element.
Second, I traversed all the elements in the given array via loop
If there is a greater element than the current maximum element, the the new current maximum becomes that element and maxIndex becomes the current i in the loop
At the end, I returned the index of the maximum element

Time complexity is O(n) because there is only one loop and I visited all the elements once
Space complexity is O(1). No extra space used apart from two variables.
